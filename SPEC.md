# Gravelle Estrie ride tracker

## About
This is a static website that consolidate the Gravelle Estrie cycling club rides

## Data source
https://www.facebook.com/groups/388978452640649/events
https://ridewithgps.com/users/4058724

Also available on Strava under this user : https://www.strava.com/athletes/89793076

## Requirements
- Running in a docker container
- A python/postgres backend with an admin console to manage the data
- Generating a static website i can publish on github pages
- Thumbnail of the ride map in the search result
- Search by name, ride start city
- Filter by ride length, ride elevation
- A detail page with the ride map and all the ride specs
- Links to Strava and RideWithGPS
- Post MVP: A "send to Garmin" button
